================================
=> Sro DDJ Viewer v0.8
================================
.Net Framework 2.0 or higher required!

=> Description:
================================
-> SroDDJ Viewer is a simple *.ddj viewer and converter.

=> Features:
================================
-> Read DDS/DDJ/DAT/BMP/ICO/GIF/JPG/PNG/PSD/TGA/TIF and many more!
-> Write DDS/DDJ/DAT/BMP/JPG/PNG/TGA (writing only compressed DXT1 DDS/DDJ textures. no transparancy. sorry about that, maybe in future version)
-> Play, build and export Silkroads bitmap animations
-> Converter for batch converting many files
-> Set as default Viewer for DDS/DDJ/DAT files
(it support all popular image formats, but you need to assign them manually, if you want them to open with Sro DDJ Viewer)
-> Previous/Next image buttons for quick browsing trough directory
-> Drag and Drop support
-> Path and FileName in Main title
